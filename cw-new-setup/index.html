Cardwatch V11R3 Installation

  • Progress Installation
  • Cardwatch Installation

Progress Installation

Locate the unzipped Progress folder, and run the setup as administrator.

Proceed with the installation.

Load the provided license file.

For all non-server installations, remove Workgroup RDBMS and ensure Personal RDBMS is the only product to be installed.

Set the destination and working paths as follows:

  • Destination directory: C:\Progressx86\OpenEdgeV117

  • Working directory: C:\CardwatchV11R3

Set the management destination and working paths as follows:

  • Destination directory: C:\Progressx86\oemgmtV117

  • Working directory: C:\CardwatchV11R3\oemgmt

Proceed with the rest of the installation.

OpenEdge Explorer can be un-checked if the installation is for registers.

Cardwatch Software Installation

Locate the unzipped CardwatchV11R3 folder, and run the setup as administrator.

Proceed with the installation.

Ensure that 11.7 is the selected version of Progress.

Set the desired user and company name, and proceed with the rest of the installation.

The installation will proceed to the Cardwatch settings setup.

Select Client for Software Type for any non-server installations.

Select the icons to create as per the device's uses, and register type if required.

Under Host (Server) Name, enter the name set up / provided by your IT.

Database Information:

Enter the port values used by your IT setup. The default values used are:

  • CDW Database: ServerCdw and Port 3011.
  • POS Database: ServerPos and Port 3021.

The REP database can be omitted if the installation is for a register-exclusive device, but is required for devices that require Back Office access.

  • REP Database: ServerRep and Port 3031.

The installation will complete at this point.
